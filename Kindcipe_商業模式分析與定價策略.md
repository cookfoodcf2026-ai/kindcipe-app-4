## 二、Mr. Cook 深度分析

### 2.1 商業模式

- **Freemium 訂閱制**
  - 免費版：15 個食譜、3 次 AI 生成、3 次掃描
  - Pro 版：$3.99/月 或 $29.99/年
- **收入引擎**：免費版限量引流 → AI 功能付費解鎖

### 2.2 收入估算

| 項目 | 保守 | 中等 | 樂觀 |
|------|------|------|------|
| 轉換率 | 2% | 3.5% | 5% |
| 付費用戶 | 2,700 | 4,725 | 6,750 |
| 月收入 | $8,100 | $14,175 | $20,250 |
| **年收入** | **$97K** | **$170K** | **$243K** |

### 2.3 行銷策略（最重要！）

| 渠道 | 佔比 | 說明 |
|------|------|------|
| **自然搜尋 SEO** | **48%** | 免費工具（食譜產生器、營養計算、份量調整）引導搜尋流量 |
| **直接流量** | 50% | 品牌知名度 + 口碑 |
| **社交媒體** | 0.3% | Instagram / X / Discord / TikTok（效果不佳） |
| **付費廣告** | 1.6% | 極少量 Google/Facebook 廣告 |
| **Chrome Extension** | — | 瀏覽器一鍵匯入食譜 |

**關鍵啟示**：Mr. Cook 幾乎不花錢買廣告，靠 SEO + 免費工具獲取 98% 流量。

### 2.4 成本結構

| 項目 | 月費估算 |
|------|---------|
| Vercel 主機 | $50–$200 |
| LLM API（最大開銷） | **$500–$3,000** |
| 資料庫 + 儲存 | $100–$400 |
| Cloudflare | $20–$200 |
| **總計** | **~$1,000–$4,500** |

## 三、Honeydew 深度分析

### 3.1 商業模式

- **Freemium 訂閱制 + 聯盟行銷**
  - 免費版：有限食譜儲存、基本功能
  - Honeydew Plus：$6.99/月 或 $39.99/年 — 無限食譜、家庭共享（6人）、無廣告
- **第二收入來源**：Instacart 導購分潤（用戶在 App 下單買菜，Honeydew 抽佣）

### 3.2 收入估算

- 假設 150K 總用戶，5–10% 轉換率
- 年收入約 **$400K–$900K USD**
- 另加 Instacart 分潤（無法估算，但為被動收入）

### 3.3 行銷策略

| 渠道 | 說明 |
|------|------|
| **App Store 優化（ASO）** | 關鍵字「recipe manager」「meal planner」「TikTok recipes」排名高 |
| **社交媒體** | Instagram (@honeydewcook) 展示社群食譜匯入功能 |
| **Chrome Extension** | 瀏覽器匯入食譜入口 |
| **口碑 / 家庭共享** | 家庭共享功能創造病毒傳播（一人付費，全家使用） |
| **內容行銷** | Ghost blog — SEO 文章 |

### 3.4 Instacart 導購模式（Kindcipe 可參考）

```
用戶排好一週餐單
      ↓
食材自動列成購物清單
      ↓
一鍵跳轉 Instacart 下單
      ↓
Honeydew 賺取佣金（約 5–8%）
```

## 八、行銷推廣策略

### 8.1 SEO + 內容行銷（抄 Mr. Cook 策略）

Mr. Cook 48% 流量來自自然搜尋。作法：

1. **建立免費 SEO 工具網頁**
   - 「一週餐單產生器」（免費網頁版）
   - 「街市食材價錢查詢」（對接消委會數據）
   - 「食材替代品查詢」
   - 每款工具 → 收集 Email → 下載 App

2. **SEO 關鍵字佈局**
   - 「IG 食譜下載 香港」「小紅書食譜保存」「YouTube 食譜儲存」
   - 「香港買餸清單 App」「每週餐單規劃」「工人姐姐煮食」
   - 「惠康優惠」「百佳特價」「街市買菜」
   - 「2024 香港最強食譜 App 推薦」

### 8.2 香港媽媽社群行銷

在香港的 Facebook Group 推廣：

- **目標群組**：香港媽媽會、親子王國、主婦聯盟
- **切入點**：
  - 「工人姐姐煮食溝通好麻煩？呢個 App 中英對照 + 自動買餸清單」
  - 「IG Reels 見到食譜，點樣 save 低？Kindcipe 一鍵匯入」
  - 「每星期排餐好頭痛？AI 幫你自動排好一週菜單」

### 8.3 WhatsApp 病毒傳播

香港家庭用 WhatsApp 溝通，邀請機制就是病毒入口：

```
陳太太：你 download 咗 Kindcipe 未呀？
          我同工人姐姐 share 緊買餸清單，超好用！
          你用唔用？我 invite 你入我個廚房！
```

每個家庭 = 至少 2–4 個成員（主婦 + 工人 + 伴侶）。

### 8.4 App Store 優化（ASO）

**關鍵字**：食譜、買餸清單、餐單規劃、煮食 App、香港食譜、工人姐姐
**語言**：繁體中文（主要）、English、Bahasa Indonesia（工人）、Filipino

### 8.5 免費工具引流（Mr. Cook 策略）

在 App 正式上線前，先做一個「一週餐單產生器」網頁：

- Free web tool：「輸入偏好 → AI 產生一週餐單」
- 用戶用完後顯示「下載 App 管理你的餐單」
- 收集 Email → App 上線時通知

## 十、你現在可以做的 3 件事

### 2. 提高定價

從規格文件的 HK$28/月 → **HK$38/月**（或開 HK$288/年），仍然比 Honeydew 平 30%。

### 3. 最低可行增長（Minimum Viable Growth）

1. ✅ 上線 App Store + Google Play
